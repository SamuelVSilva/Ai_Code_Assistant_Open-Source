# AI Code Assistant - V3.0.2-alpha

Um assistente de código inteligente com interface inspirada no VS Code e Cursor.

## Novidades da V3.0.2
- **Interface Premium**: Nova interface construída em PyQt6 com visual moderno e cinza escuro.
- **Activity Bar**: Acesso rápido a ferramentas (Explorer, Pesquisa, etc).
- **Terminal Integrado**: Terminal persistente para execução de comandos.
- **Chat Moderno**: Sistema de chat com bolhas e suporte a múltiplas IAs.
- **Build para Linux**: Suporte oficial para distribuição em Linux/Debian.

## Como Executar
1. Instale as dependências: `pip install -r requirements.txt`
2. Execute: `python src/main.py`
